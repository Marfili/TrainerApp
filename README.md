# TrainerApp

TrainerApp is a calendar designed to help a trainer organize appointments and payments with trainees throughout the month.  
Built with JavaScript using the React Native framework in VS Code ,with AI-assisted development for code generation and debugging.
The app is simple and easy to use,with a clean interface where you can add,edit, and view training sessions and trainees. It was built as a practice project to improve my skills in React and understand how to build interactive web applications.

# Features
- Schedule and track appointments with trainees.
- Manage trainee records.   
- View monthly calendar of sessions and payments.  
- Add, edit,or remove trainee information.  


# Technologies

-JavaScript

-React Native

-Expo

-AsyncStorage

-React Navigation


The app was developed using AI-assisted coding (Claude),but all product decisions,logic design,testing,deployment and debugging were handled manually to ensure full functionality.
